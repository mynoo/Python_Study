### 내장함수 (built-in function)
   파이썬 내장 함수들은 외부 모듈과는 달리 import를 필요로 하지 않는다.
  
### str()
- 문자열 형태로 객체를 반환하여 리턴하는 함수이다.
   ```
>>> str(3)
'3'
>>> str('hi')
'hi'
>>> str('hi'.upper())
'HI'
```
